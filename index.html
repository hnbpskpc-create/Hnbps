  3. ចុច Deploy > New Deployment រួចយកប្រភេទ Web App (កំណត់ Who has access to 'Anyone').
  4. ចម្លងតំណភ្ជាប់ Web App URL នោះមកដាក់ក្នុងប្រអប់ខាងលើ រួចចុច 'រក្សាទុក' និង 'Sync'។
